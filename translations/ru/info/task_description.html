Квадратные корни, кубические корни, корни четвёртой степени... Всё это слишком скучно для Николя. Ему нужно найти супер корень! С вашей помощью он почти наверняка его найдёт.

Супер корень числа N — это такое число x, при котором xx = N.

Результат должен быть точен как xx ≈ N±0.001. Или N − 0.001 < xx < N + 0.001.

Входные данные: Число (N), как целое число.

Выходные данные: Супер корень (x), как число с плавающей точкой или целое число.

Пример: 

super_root(4) == 2
super_root(27) == 3
super_root(81) == 3.504339593597054

Как это используется: Этот принцип может быть полезен в криптографии. А вы посмотрите как работает ваш калькулятор вычисляя корни.

Предусловия:
1 ≤ number ≤ 10 ** 10
